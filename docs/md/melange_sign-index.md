## melange sign-index

Sign an APK index

### Synopsis

Sign an APK index.

```
melange sign-index [flags]
```

### Examples

```
  melange sign-index [--signing-key=key.rsa] <APKINDEX.tar.gz>
```

### Options

```
  -h, --help                 help for sign-index
      --signing-key string   the signing key to use (default "melange.rsa")
```
